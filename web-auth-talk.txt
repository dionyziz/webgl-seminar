1η ώρα
======

Τα φώτα σβήνουν, παίζει το Rome full σε πλήρη οθόνη για 2 λεπτά. Τα φώτα ανάβουν,
εμφανιζόμαστε και τους δείχνουμε ότι είναι website και όχι desktop
και επίσης ότι δεν είναι video. Σκάμε μύτη με καπέλα άγιου βασίλη.

Εξηγούμε τι είναι τα γραφικά υπολογιστών γενικά.
Εξηγούμε τι είναι τα γραφικά υπολογιστών μέσα σε ιστοσελίδες.
Διαφορά native rendering και plugins.
Επιτάχυνση υλικού VS επιτάχυνση λογισμικού.

Συστηνόματε και λέμε τι έχουμε κάνει στη ζωή μας.
Σύνοψη και στόχοι του μαθήματος:
- Να σας δείξουμε την ιδέα πίσω από τα γραφικά στους υπολογιστές
- Απλές ιδέες με όμορφο αποτέλεσμα
- Πώς ανεβαίνουμε διαστάσεις: 0, 1, 2, 3
- Ξεκινάμε από τα βασικά συστατικά στοιχεία των γραφικών και πάμε σε όμορφους κόσμους
- Από ένα σημείο μέχρι ένα περιβάλλον ενός παιχνιδιού
- Εφέ με σωματίδια: Φωτιά, καπνός
- Απομυθοποίηση του πώς φτιάχνονται τα γραφικά σε παιχνίδια και άλλες εφαρμογές
- Ενθάρρυνση να γίνουν δημιουργικοί και να εμπνευστούν

Αναφέρουμε ότι η διάλεξη βιντεοσκοπείτε και ότι όλο μας το υλικό είναι Creative Commons και MIT.

W = ( A + B + C ) / 3
I = max( 0, ^l . ^n )
R' = I * R
G' = I * G
B' = I * B

Μετασχηματισμός: Η περιστροφή σε 2 διαστάσεις. ("Αντίστοιχα γενικεύεται και στις 3")
Εξήγηση στο μαυροπίνακα.

r = sqrt( x^2 + y^2 )
φ = atan2( y, x )
r' = r
φ' = φ + θ
x' = r'cosφ'
y' = r'sinφ'
